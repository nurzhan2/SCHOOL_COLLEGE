## КАК РАБОТАЕТ ДЕПЛОЙ НА RAILWAY

1. В папке есть `package.json`:
```json
{
  "name": "shkolledge-site",
  "scripts": { "start": "serve . -p $PORT --no-clipboard" },
  "dependencies": { "serve": "^14.2.0" }
}
```

2. И `railway.json`:
```json
{
  "deploy": { "startCommand": "npx serve . -p $PORT --no-clipboard" }
}
```

Railway видит package.json → устанавливает Node.js → запускает serve → сайт работает.

**Если Railway не поднимается:** проверь логи деплоя, убедись что `package.json` 
в корне репозитория (не в подпапке).
